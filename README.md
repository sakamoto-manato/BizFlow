# BizFlow

経営管理ダッシュボード（インボイス制度対応）

## 機能

- **ダッシュボード**: 累計入金、未回収額、在庫アラート、インボイス対応状況
- **取引先管理**: 顧客一覧、T番号・適格/免税バッジ、検索・フィルタ
- **債権管理**: 未回収・延滞・入金済の管理
- **請求書/見積書作成**: 8%/10%税率切替、T番号自動付与、プレビュー
- **在庫管理**: 品目一覧、カテゴリフィルタ、在庫ステータス
- **入金管理**: 入金概要、インボイス仕分け（適格/免税）
- **給与明細**: 支給・控除内訳、手取り推移

## セットアップ

```bash
npm install
npm run dev
```

開発サーバー: http://localhost:5173/
